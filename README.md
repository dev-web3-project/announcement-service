# Announcement Service

Ce microservice gère la communication globale de la plateforme LMS (Annonces, Notifications, etc.).

## Fonctionnalités
- Publication d'annonces
- Système de notifications système

## Exécution
Ce service est conçu pour fonctionner au sein de l'infrastructure `core`.
1. Assurez-vous qu'il est cloné dans le dossier `lms-microservices/announcement-service` du projet Core.
2. Lancez `./run-local.sh` depuis le projet Core.

Le service tournera sur le port `8086`.
